*上面提到的其中一个坑说明：*

当执行修改密码的时候出现 ： database not open

提示数据库没有打开，不急按如下操作

输入：alter database open;

注意了：这里也许还会提示 ： ORA-01507: database not mounted

解决办法：

输入：alter database mount;

输入 ：alter database open;

然后就可执行 修改数据库密码的命令了

改完之后输入：ALTER PROFILE DEFAULT LIMIT PASSWORD_LIFE_TIME UNLIMITED;
