[1] alt + insert 创建内容
[2] psvm main方法
[3] sout 输出语句
[4] Ctrl + d 复制一行
[5] Ctrl + y 删除一行
[6] Ctrl + Shift + Up/Down 代码向上/下移动
[7] Ctrl + n 搜索类
[8] alt + insert 生成代码

[12] fori 回车即可 for循环
[13] ctrl + alt + t 代码包围try-catch,if,while等
[14] ctrl + alt + -> 向后 ctrl + alt + <- 向前
[15] 显示代码结构：alt + 7
[16] 显示导航栏：alt + 1
[17] 撤销 ctrl + Z
[18] REDO 操作 ctrl + shif + z
[19] Tab 缩进 shift+Tab 撤销缩进
