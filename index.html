python的闭包与修饰器

发表于 2021-11-10 分类于 闭包与修饰器

闭包

• 将函数作为返回值也是高阶函数我们也称为闭包
• 1.闭包的好处
• 通过闭包可以创建一些只有当前函数能访问的变量
• 可以将一些私有数据藏到闭包中
• 2.行成闭包的条件
• 函数嵌套
• 将内部函数作为返回值返回
• 内部函数必须要使用到外部函数的变量或者参数

装饰器

通过装饰器，可以在不修改原来函数的情况下来对函数进行扩展
在开发中，我们都是通过装饰器来扩展函数的功能的

lambda匿名函数

  • 语法：lambda 参数：表达式
  • print(lambda a,b :a+b)(10,20))
  • nonlocal a 提示a 值是作用域外的

Python 学习指南

平台：linux 平台

系统：线程、进程、Shell编程

网络: 网络原理、HTTP协议。Socket开发、IO多路复用、异步IO开发

算法要求：斐波那契数列、求指数、常见排序算法等，甚至要求现场写

数据库：关系数据库MySQL，NoSQL应该了解，最好一个

中间件：消息队列原理和应用，例如RabbitMQ

WEB开发：偏后端，MVC框架要求会Django、Flask之一

可视化：WEB的前端多一些HTML、CSS、JS要会

数据分析和AI：要求掌握数据分析的数据理论，机器学习常用库使用
